# Security Policy

## Supported versions

The latest published release is the only supported version. Older builds are not patched.

## Reporting a vulnerability

If you have discovered a security issue, please do not open a public GitHub issue.

Email the maintainer privately at `seanreid.mail@gmail.com` with:

- A clear description of the issue
- Steps to reproduce
- Affected platform and version
- Any proof-of-concept code or logs

You can expect an acknowledgement within 72 hours. Disclosure timelines will be agreed once the report is triaged.

## Scope

In scope:

- The game client binary distributed via GitHub Releases.
- The backend Workers and Durable Objects hosted on Cloudflare.
- The project website hosted on GitHub Pages.

Out of scope:

- Third-party services we depend on. Report those upstream.
- Self-hosted or modified forks of this project.
